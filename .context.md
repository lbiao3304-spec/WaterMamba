## 简介

WaterMamba 是一个基于 **Mamba（State Space Model）** 的水下图像增强网络。核心算子 **SS2D** 在特征图 4 个方向上做选择性扫描（原图、翻转、转置、翻转转置），替代 Self-Attention，以 **线性复杂度** 实现全局感受野建模。网络采用 4 级编码器-解码器 U-Net 结构，配合 skip connection 与 DWT/IDWT 无损下采样实现端到端水下图像增强。

**关键特性：**
- 🔁 **SS2D (4-Directional Selective Scan)** — 替代 Self-Attention，线性复杂度 O(N)
- 🌊 **SCOSSBlock** — 基础构建块，LN → SS2D → DropPath → 残差连接
- 🌀 **DWT/IDWT 下采样** — 离散小波变换无损下采样，完整保留空间信息
- 📐 **Tile Test 分块推理** — 8GB 显存即可处理任意尺寸大图

## 模型架构

### SCOSSBlock（基础构建块）

SCOSSBlock 是 WaterMamba 的核心模块，相当于 Transformer Block，但将 Self-Attention 替换为 SS2D：

```
架构：LayerNorm → SS2D（4方向选择性扫描） → DropPath → 残差连接
```

```python
class SCOSSBlock(nn.Module):
    def __init__(self, hidden_dim=48, d_state=16, expand=2., d_conv=4):
        self.ln_1 = nn.LayerNorm(hidden_dim)      # 稳定状态空间数值
        self.SOSS = SS2D(
            d_model=hidden_dim,                   # 特征维度
            d_state=d_state,                      # SSM 状态维度
            expand=expand                         # 通道扩展系数
        )
        self.drop_path1 = DropPath(drop_path)     # 随机深度正则化

    def forward(self, input):
        # [B, C, H, W] → [B, H, W, C]（适配 Mamba 序列格式）
        x = input.permute(0, 2, 3, 1)
        x = self.ln_1(x)
        x = self.SOSS(x)                          # SS2D 全局建模
        # 残差连接 + DropPath
        x = input.permute(0, 2, 3, 1) + self.drop_path1(x)
        return x.permute(0, 3, 1, 2)
```

### DWT/IDWT 无损下采样

```python
class Downsample_DWT(nn.Module):
    """DWT 分解：低频 + 3 个高频分量 → 1×1 卷积"""
    def forward(self, x):
        yl, yh = self.dwt(x)
        return self.conv(torch.cat([yl, yh]))

class Upsample_IDWT(nn.Module):
    """IDWT 合成：分离低频 + 高频 → 小波逆变换"""
    def forward(self, x):
        yl, yh = split(x)
        return self.idwt((yl, yh))
```

DWT 将图像分解为 LL/LH/HL/HH 四个频带，完整保留空间信息，下采样不丢失高频细节。

### Tile Test 分块推理

大图自动分块推理 + 高斯权重融合，8GB 显存也能处理任意尺寸图像：

```
全图输入 → pad_test 补齐 → tile_test 分块推理 → 高斯融合缝合 →
pad_test 裁剪 → 输出与输入同尺寸 → 与 GT 计算 PSNR/SSIM
```

- 分块大小：256 × 256（overlap 32px）
- DWT 要求输入尺寸能被 8 整除（`window_size: 8`），pad_test 自动补齐

| `{img_name}_gt.png` | Ground Truth 参考图 |
